# OOP_lab_work_02
The program is in C# 

Task 1:
In a one-dimensional array consisting of N real elements, calculate:
• sum of array elements with odd elements;
• the sum of array elements that are located between the first and last negative elements.
Move the first M elements to the end of the array (M is entered from the keyboard, M<N).

Task 2:
A text line is entered from the keyboard. Make a program that
• counts the number of words in the text;
• displays a word on the screen, displays the number of vowels;
